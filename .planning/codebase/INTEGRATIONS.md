# Integrations

## External Services & APIs
- **Web Speech API**: Used for voice dictation and Text-to-Speech (TTS) readback confirmation.
- **react-webcam**: Integrated for document and ID scanning directly from the frontend.
- **jsPDF**: Used for generating PDF receipts and invoices.
- **react-transliterate**: Used for Indic language transliteration support.
- **Axios**: HTTP client for communicating with the backend API.
