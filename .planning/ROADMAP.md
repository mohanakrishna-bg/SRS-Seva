### Phase Details

### Phase 1: Core Devotee & Seva
Goal: Establish the foundation for devotee records and booking sevas.
**UI hint**: yes
Requirements: DEV-01, SEV-01, SEV-02
Success criteria:
1. Staff can create and edit a devotee profile with gotra/nakshatra.
2. Staff can book a specific seva on a future date for a devotee.
3. Sevas can be marked as fulfilled/completed.

### Phase 2: Financials & Accounting
Goal: Implement receipt generation, tracking, and ledgers.
**UI hint**: yes
Requirements: FIN-01, FIN-02, FIN-03
Success criteria:
1. System generates a downloadable PDF receipt for a transaction.
2. Staff can log a manual UPI/Cash/Cheque payment.
3. Daily totals for receipts and expenditures are displayed.

### Phase 3: Inventory System
Goal: Build separated tracking for Fixed Assets and Consumables.
**UI hint**: yes
Requirements: INV-01, INV-02, INV-03
Success criteria:
1. UI has distinct screens/workflows for Fixed Assets vs Consumables.
2. Staff can add and edit items in both categories.
3. A Google Photos URL can be linked and displayed for inventory items.

### Phase 4: Reporting & Insights
Goal: Add interactive drill-down reports for all modules.
**UI hint**: yes
Requirements: REP-01, REP-02, REP-03
Success criteria:
1. Drill-down reports available for Devotees and Sevas.
2. Drill-down reports available for Financial ledgers.
3. Drill-down reports available for Fixed Assets and Consumables.

### Phase 5: Public Portal
Goal: Launch the internet-facing portal with bilingual UI.
**UI hint**: yes
Requirements: PUB-01, SYS-01, SYS-02
Success criteria:
1. A public-facing route exists displaying available sevas.
2. Devotees can initiate a booking/donation via the public portal.
3. Interface text can be toggled between English and Kannada.
4. Voice dictation is fully functional in input fields.
